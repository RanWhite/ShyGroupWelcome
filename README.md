SkyWelcome是一款利用小天机器人  
实现用户进群自动欢迎的插件  
运行该插件需要安装Java环境   

如何更改欢迎语  
打开目录下文件夹/group  
找到对应群wxid.json  
修改文件中的“欢迎语”  
%nick%将自动替换为它的昵称
